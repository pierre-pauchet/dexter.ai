# Dexter.ai

Dexter.ai is an advanced AI-powered tool designed to analyze blood test results and provide personalized advice. It leverages a fine-tuned version of DeepSeek 8B to process and interpret medical data with precision.

## Features

- **OCR Integration**: Dexter.ai extracts data from blood test images or PDFs using Optical Character Recognition (OCR) technology.
- **Data Formatting**: Automatically organizes and formats the extracted data for easy interpretation.
- **Personalized Advice**: Provides tailored health insights and recommendations based on the blood test results.
- **User-Friendly Interface**: A web-based front end hosted on the user's machine offers an intuitive chatbot interface for seamless interaction.


## How It Works

1. Upload a blood test image or PDF.
2. Dexter.ai processes the file using OCR to extract relevant data.
3. The data is analyzed and formatted for clarity.
4. Receive actionable advice and insights through the chatbot interface.

## Getting Started

To use Dexter.ai, follow these steps:
1. Clone the repository.
2. Install the required dependencies.
3. Run the application locally to access the web interface.
4. Upload your blood test and start receiving insights.

## Model Weights

The fine-tuned DeepSeek 8B model weights used in Dexter.ai can be accessed on [Hugging Face](https://huggingface.co/).
## Technologies Used

- **DeepSeek 8B**: Fine-tuned model for medical data analysis.
- **Pytesseract-OCR**: For extracting text from images and PDFs.
- **Web Application**: Local hosting for a responsive chatbot interface.
